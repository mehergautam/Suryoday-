## 📖 Overview

**Suryoday** is a modern wellness application that guides users through the complete **12-step Surya Namaskar** sequence using a premium, distraction-free interface.

The goal is to make yoga feel as engaging and polished as modern fitness applications by combining guided practice, session customization, streak tracking, workout statistics, and progress visualization.

---

## ✨ Features

- 🌄 Premium dark UI inspired by modern wellness apps
- 🧘 Guided 12-step Surya Namaskar flow
- ⏱ Adjustable pose duration
- 🔁 Custom number of sets
- ⏳ Rest between sets
- 🚀 Countdown before session
- 🎯 Daily goal tracking
- 📊 Workout statistics dashboard
- 📅 Contribution calendar
- 🔥 Streak tracking
- 🏆 Achievement badges
- 📜 Workout history
- 🔊 Voice guidance
- 🎵 Ambient meditation audio
- 💾 Local progress storage
- 📱 Fully responsive design
